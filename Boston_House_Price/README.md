# Boston House Price Prediction

![Python](https://img.shields.io/badge/Python-3.8+-blue?logo=python&logoColor=white)
![Scikit-Learn](https://img.shields.io/badge/Machine_Learning-Scikit_Learn-F7931E?logo=scikit-learn&logoColor=white)

This project focuses on predicting the median value of owner-occupied homes in the Boston area using classic Machine Learning regression algorithms on a well-known historical dataset.

## Overview
- **Problem:** Predicting continuous target variables based on features like crime rate and tax rates.
- **Approach:** Extensive EDA, feature scaling, and regression models (Linear Regression, Random Forest).
- **Evaluation:** Evaluated based on R-Squared (R2) and RMSE.

## Tech Stack
- **Data Manipulation:** Pandas, NumPy
- **Machine Learning:** Scikit-Learn
- **Visualization:** Seaborn, Matplotlib
